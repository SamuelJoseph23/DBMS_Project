# DBMS_Project

A simple project made using **C#** and **MySQL** to demonstrate a basic Windows Forms application with a database backend.

---

## Features

- Built using C# (.NET Framework, WinForms)
- Interfaces with a MySQL database
- Multi-form GUI structure (`Form1.cs` through `Form10.cs`)
- Readable, extensible code
- Modular design with designer files and resource files for each form

---

## Project Structure

- `DBMS_Project.sln`: Visual Studio Solution file
- `DBMS_Project.csproj`: C# project file
- `Program.cs`: Entry point for the application
- `FormX.cs`, `FormX.Designer.cs`, `FormX.resx` : Source code, designer, and resource files for Forms 1 through 10
- `.gitattributes`, `.gitignore`: Repository and environment config
- `README.md`: Project overview

---

## Getting Started

### Prerequisites

- Windows OS
- Visual Studio with C# development tools
- MySQL server/database access

### Setup Instructions

1. **Clone the Repository**
git clone https://github.com/SamuelJoseph23/DBMS_Project.git

2. **Open in Visual Studio**
- Open `DBMS_Project.sln`
3. **Configure Database Connection**
- Update connection strings in the code to point to your MySQL database.
4. **Build and Run**
- Press F5 or click "Start" to build and launch the application.
